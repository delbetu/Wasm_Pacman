## Ruby Pacman Clone Running on WebAssembly

This is an unfinished version of the game.
Anyone is welcome to contribute.

![demo](wasm_pacman.gif)

## Running the app
Run a http server on this directory:

`ruby -run -e httpd . -p 8080`
